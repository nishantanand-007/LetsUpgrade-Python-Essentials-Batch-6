Q2. What are code environments? Types?
Ans
Code environment is a place where we can writecode, compile it, and see the output/outcome all together in just one place.
There are 3 types :
1. Text editors - VS code , ATOM
2. Full IDE's - PyCharm , Spyder
3. Notebook Environment - Jupyter Notebook
